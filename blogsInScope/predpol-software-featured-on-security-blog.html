PredPol Software Featured on Security Blog

Posted on Sep 19, 2012 in Chameleon Security, Dr. Jeff Brantingham, Los Angeles Police, Predictive Policing, PredPol, UCLA | One Comment

The PredPol software program was featured on the Chameleon Security blog today. Check out the article below:

For ages, law enforcement officers have plotted crimes on maps as a way of tracking activity and focusing attention on critical areas.  In the “old days”, color-coded push pins were used to manually mark activity in each given sector.  The latest tool takes this basic analytic model a step or two further. 

Pred Pol – Predictive Policing – uses the same kind of algorithms used by seismologists to attempt to predict aftershocks.  It was developed by a team that included mathematicians, criminologists and anthropologists.  In use at five Los Angeles Police Department divisions, the system seems to be helping.  Crime has dropped off between 12 and 27 percent in areas where Pred Pol has been implemented.

The system developer Jeff Brantingham of UCLA explains that data from repeat crime victims is constantly re-calibrated which produces dynamic, graphic patterns that go beyond what humans can produce.  Although I believe we too often undervalue the power of human intuition and human analytics, this is an interesting and powerful system.  I look forward to see where it goes.
